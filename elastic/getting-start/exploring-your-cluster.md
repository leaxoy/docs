# 探索集群


REST API

现在，我们启动和运行了我们自己的节点（和集群），下一步是了解如何与它进行通信。幸运的是，Elasticsearch提供了一个非常全面和强大的REST API，你可以用它来与集群交互。期间，可以使用API来完成如下几件事情：
- 检查您的群集，节点和索引的健康，状态和统计信息
- 管理你的集群，节点和索引数据和元数据
- 执行CRUD(创建，读取，更新和删除)，并针对您的索引搜索进行操作
- 执行高级搜索操作，如分页，排序，筛选，脚本，聚合，还有许多其他的事情
